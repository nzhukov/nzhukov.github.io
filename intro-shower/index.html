Недостатки Shower

Недостатки

  1. Высокий порог входа.
  2. Презентация состоит из > 1 файла.
  3. Удобство размещения на слайде элементов.
  4. Ограниченное количество элементов «из коробки».

Например: темы, стили, шрифты и прочее.

Достоинства Shower

Достоинства

  1. Может быть размещена онлайн.
  2. Открывается с помощью любого современного браузера.
  3. Экспорт в формат PDF.
  4. Интеграция в презентацию «сложного» контента.
  5. Интеграция в LMS (SCORM).
  6. Могут быть встроены опросы, тесты (в том числе «Формы Google»).
  7. Презентации индексируются поисковыми системами.

Цели создания презентации с Shower

Цель №1

Существует необходимость создать презентацию с контентом, индексируемым поисковыми системами, с которой возможно было бы работать на любых платформах (в том числе и на мобильных) без установки какого-либо специализированного ПО и чтобы возможно было использовать такую презентации в электронных образовательных ресурсах.

Цель №2

Встраивать внутрь презентации нестандартные объекты. Например: опрос, мультимедиа-контент, интерактивные элементы.

Возможности

Подробнее о возможностях

  1. Основан на HTML, CSS, Javascript и расширяем с помощью этих языков.
  2. Работает во всех современных браузерах.
  3. Поддерживает 2 темы для оформления презентаций.
  4. Экспорт презентации в PDF формате.
  5. Виды контента в презентации из стандарта HTML.
  6. Форматирование и его расширение.

Жизненный цикл

Основные этапы ЖЦ презентации

  1. Начало работы — загрузка образца.
  2. Вставка контента и форматирование содержимого.
  3. Публикация презентации или экспорт в PDF.
  4. Внесение изменений.

Этапы создания

Как создать презентацию с Shower?

  1. Загрузка шаблона.
  2. Разархивировать и открыть index.html.
  3. Слайд — это тег section с классом slide.
  4. Редактирование кода index.html.
  5. Вставка изображения, таблицы.
  6. Вставка видеоролика (тег iframe или video).
  7. Экспортировать в PDF.
